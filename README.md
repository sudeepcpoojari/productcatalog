# productcatalog
Mini Product Catalog Web App  Assignment

## Overview
A simple Django web app to manage a list of products.  
Users can view, add, and search products.

## Tech Stack
1.Python (Django)
2.SQLite
3.HTML + CSS

## How to Run
1. Clone the repo  
2. Run 'python manage.py migrate' 
3. Run 'python manage.py runserver' 
4. Open browser - 'http://127.0.0.1:8000/'

## Features
1.View product list  
2.Add new products  
3.Search products by name  


## Challenges Faced
1.Designing clean structure
2.Implementing search logic using `Q` objects
